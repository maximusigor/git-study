Git with CMD

  • git clone path-to-repository (копирует с github созданный репозиторий на локальный компьютер)
  • git status (статус текущего локального репозитория)
  • git add path(name)-to-file (добавить новый файл под контроль git)
  • git add . path(name)-to-file (добавить все новые файлы под контроль git)
  • git commit -am "комментарий к изменениям кода, файлов и т.д."
  • git push (отправить все закомиченные изменения в репозиторий на сервер)
  • git pull (загрузить новые изменения с сервера)
